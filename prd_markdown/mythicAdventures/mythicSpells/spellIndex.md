**Mythic** : Some of these spells have mythic versions just as preexisting spells like _ [animate dead](spells/animateDead.md#_animate-dead)_ and _ [fireball](spells/fireball.md#_fireball)_ do. The information for the mythic version is included in the new spell's description.

**Augmented** : If a spell has an augmented version, it's listed in the spell description.
